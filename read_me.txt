This is an application to handle user's training sessions.

How to run:
python app.py


Files:
- app.py            - the entry point of the program
- interface_utils   - all the functions for interface
- models.py         - db tables definitions
- utils.py          - functions to interact with db

- clean.py          - script to reset db

- interface_test.py - unittests for funcitons from interface_utils
- db_test.py        - unittest for functions from utils.py
